## 📂 Featured Projects

### 1. [ServisCiragim](./ServisCiragim) — *Automotive SaaS Platform*
A multi-tenant SaaS architecture designed for automotive service stations and spare parts dealers.
* **Highlights:** Advanced JWT-based RBAC, token versioning/blacklisting, automated stock difference algorithms, and Nilvera e-Invoice API integration.

### 2. [CekiciAsistance](./CekiciAsistance) — *Tow Truck & Dealer Management ERP*
An end-to-end B2B logistics and dispatch platform managing tow truck networks and dealer sales.
* **Highlights:** Real-time incident dispatching with geolocation, Garanti Bank 3D Secure Virtual POS integration, and automated financial commission reconciliations.

### 3. [KayaAssistance](./KayaAssistance) — *Roadside Assistance Platform*
A centralized web application managing 24/7 roadside assistance operations, package sales, and dealer networks.
* **Highlights:** Secure external API bridges, dynamic contract PDF generation, session-based security, and robust IP-based rate limiting.

### 4. [PatienceRush](./PatienceRush) — *Android 2D Platformer Game*
A fast-paced mobile game built entirely from scratch utilizing Java and the native Android SDK.
* **Highlights:** Custom axis-separated 2D physics engine (built without third-party libraries), procedurally parameterized map generation, and Firebase/Google Play Games integration.
